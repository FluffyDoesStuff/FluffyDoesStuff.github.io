<h1>Installation:</h1>
<p><strong><span style="color: #ffffff;">Tested and supported Optifine versions: 1.8.9 - 1.16.5 Most stable Optifine version: 1.14.4</strong><br />1. Download and install&nbsp;<strong><a href="https://www.minecraft.net/" target="_blank" rel="noopener">Minecraft,</a></strong>&nbsp;setup your game profile of the version you want to run.<br />2. Download and run the <strong><a href="https://optifine.net/downloads" target="_blank" rel="noopener">Optifine</a></strong> jar file and click the Install button. It will install a new Optifine profile in your minecraft launcher.<br />3. Run the minecraft launcher and select the newly created Optifine profile.<br />4. Start the game.<br />5. Go to Options -&gt; Video Setings -&gt; Shaders and press the "Shaders Folder" button at the bottom left of the screen.<br />6. Download the ShaderPack and place it in the newly opened window.<br />7. Select the ShaderPack you downloaded from the list ingame and enjoy!</p>
